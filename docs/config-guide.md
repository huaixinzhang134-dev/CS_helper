# 游戏配置说明

## 一、选手状态 (player.status)

| 状态 | 含义 | 可用难度 |
|------|------|---------|
| `active` | 现役选手（有队伍） | 极简 / 简单 / 困难 |
| `coach` | 教练（无论之前是否退役） | 极简 / 简单 / 困难 |
| `free_agent` | 自由人（未退役但无队） | 困难 专属 |
| `retired` | 已退役 | 地狱 专属 |
| `unknown` | 数据来源未确定 | 地狱 专属 |

## 二、难度分级

| 难度 | 英文标识 | 范围 |
|------|---------|------|
| 🌱 极简 | `trivial` | status 为 active/coach 且队伍在 team_ranking 前 30 |
| 🌟 简单 | `easy` | status 为 active/coach 且队伍在 team_ranking 前 60 |
| ⚔️ 困难 | `hard` | status 为 active/coach/free_agent（排除退役选手） |
| 💀 地狱 | `hell` | 所有选手无限制（包含退役选手） |

## 三、游戏模式

| 模式 | 说明 | 尝试次数 |
|------|------|---------|
| 🎯 个人练习 | 独自挑战，随机选目标选手 | 无限次 |
| 👥 好友PK | 邀请微信好友，比拼谁先猜出 | 每人 8 次 |

## 四、猜测反馈规则

### 战队
- 🟢 **正确** — 猜测选手的当前战队与目标选手相同
- 🟡 **接近** — 猜测选手的当前战队是目标选手的曾属战队
- ⚪ **错误** — 其他情况

### 国家
- 🟢 **正确** — 国家名称相同（中文）
- 🟡 **接近** — 赛区相同但国家不同（如 Europe / Americas / Asia）
- ⚪ **错误** — 赛区不同

### 年龄
- 🟢 **正确** — 年龄完全相同
- 🟡 **接近** — 年龄差 ≤ 2 岁（附带 ↑↓ 方向提示）
- ⚪ **错误** — 年龄差 > 2 岁

### Major次数
- 🟢 **正确** — 次数完全相同
- 🟡 **接近** — 次数差 ≤ 2（附带 ↑↓ 方向提示）
- ⚪ **错误** — 次数差 > 2

### 位置
- 🟢 **正确** — 位置相同
- ⚪ **错误** — 位置不同

## 五、提示功能

- 条件：单局猜测次数 > 6 次后可启用
- 限制：每局限用 1 次
- 内容：随机展示目标选手的 战队 / 国家 / 年龄 / Major次数（不含姓名）

## 六、数据来源

- **选手数据**：HLTV.org（通过爬虫定期同步）
- **赛事数据**：5eplay.com（每 30 分钟同步）
- **战队排名**：HLTV.org（每周同步）
